STEP OF RUNNING AXIOS 

1: For working in terminal 
Run command
1: npm init (makesure [nodemodules(Folder) +package.json+package-lock.json] is created)
1: npm install axios 
2: add a statment in js file 
"const axios = require('axios');"




2: For running on browser
Add CDN refrence in HTML file
"<script src="https://cdn.jsdelivr.net/npm/axios/dist/axios.min.js"></script>"
